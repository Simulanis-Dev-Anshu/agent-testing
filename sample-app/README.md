# Task Pulse

Small HTML/CSS/JS task board used to test the AI review bot on pull requests.

## Files

- `index.html` — page layout and task list UI
- `styles.css` — responsive styling
- `app.js` — add, complete, delete, and search tasks

## Preview

Open `index.html` in your browser.

## PR test

Commit this folder on the `test-ai-review` branch and open a PR to trigger the review workflow.
